## Qué es este proyecto
Proyecto que permite planeacion del dia dia, semana y mes familiar.
stack basado en nextjs, supabase y desplegado en vercel

## Reglas que SIEMPRE aplican
- Idioma de código: español
- No crear archivos nuevos sin confirmar primero
- Siempre correr `npm run lint` antes de dar por terminado
- Tests obligatorios para lógica nueva 

## Comandos clave
- Dev: `npm run dev`
- Test: `npm test`
- Lint: `npm run lint`
